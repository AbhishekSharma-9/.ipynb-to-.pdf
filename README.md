A lightweight, purely browser-based web application that allows users to seamlessly convert Jupyter Notebook (`.ipynb`) files into PDF format. Because it runs entirely on the client side, your data remains private and secure—no files are uploaded to any external servers.

## <picture>   <source srcset="https://fonts.gstatic.com/s/e/notoemoji/latest/1f31f/512.webp" type="image/webp">  <img src="https://fonts.gstatic.com/s/e/notoemoji/latest/1f31f/512.gif" alt="🌟" width="20" height="20"> </picture> Features

- **100% Client-Side:** No backend server required. Your files never leave your computer, ensuring complete privacy.
- **Fast & Efficient:** Immediate conversion powered by JavaScript.
- **User-Friendly Interface:** Clean, simple, and intuitive design for hassle-free conversions.
- **Zero Installation:** Works straight out of your web browser without needing Python, Jupyter, or LaTeX installed on your system.

2. Click on the upload button and select your desired `.ipynb` file.
3. Click the **Convert** button.
4. The generated PDF will automatically be downloaded to your device.

## 💻 Running Locally

If you want to run this project on your local machine, contribute, or modify it, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/AbhishekSharma-9/.ipynb-to-.pdf.git](https://github.com/AbhishekSharma-9/.ipynb-to-.pdf.git)
   ```

2. **Navigate to the directory:**
   ```bash
   cd .ipynb-to-.pdf
   ```

3. **Open the application:**
   Simply double-click `index.html` to open it in your default web browser. Alternatively, you can use a local development server like VS Code's "Live Server" extension for a better development experience.
